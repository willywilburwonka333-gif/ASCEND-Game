# ASCEND

A bright creature-raising educational RPG where real learning restores a fractured fantasy world.

## First playable slice

- Egg-first opening
- Companion naming
- Free four-direction movement
- Companion following
- NPC interaction
- Decimal bridge-restoration quest
- Guardian encounter

## Run locally

```bash
npm install
npm run dev -- --host 0.0.0.0
```
